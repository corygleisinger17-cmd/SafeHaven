# ADR-001

Human approval is always required before classification.

Status:
Accepted

Reason:

SafeHaven is an inventory system, not an autonomous decision maker.

Automatic classification can create cascading data integrity problems.

All classifications require explicit user approval.

# ADR-003

Deterministic logic precedes AI.

Status

Accepted

Reason

Deterministic classification is fully explainable and testable.

AI enhances suggestions but does not replace deterministic logic.
Users should evaluate manufacturer and item type independently.

A blended confidence score hides uncertainty and encourages overtrust.
